# gomantics.dev

Documentation site for all [gomantics](https://github.com/gomantics) libraries.

## Libraries

- **[chunkx](</(docs)/chunkx>)** - AST-based code chunking for Go
- **[cfgx](</(docs)/cfgx>)** - TOML config to strongly-typed Go code generator
- **[sx](</(docs)/sx>)** - String extensions for Go
- **[semantix](</(docs)/semantix>)** - Semantic code search with MCP integration

## Development

- Install [mise](https://mise.jdx.dev/installing-mise.html)

```bash
mise install
pnpm i
pnpm dev
```
